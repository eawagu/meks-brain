## Summary

A two-page operational reference document for [[Moniepoint]]'s [[AWS]] Enterprise Support engagement. It covers how to raise support cases, severity classification, target response times, best practices for case submission, and the escalation path from support case → TAM → Enterprise Support Manager.

## Key Points

- Support cases are raised via AWS Support Center; severity levels range from General Guidance (24h response) to Business-Critical System Down (15 min response)
- Best practice: one case per event, raised in the correct AWS account, with detailed who/what/when/where/why/how information
- For Business-Critical/Production System Down cases, use Chat or Phone and provide a conference bridge
- Escalation path: Submit case → Contact TAM → Contact Enterprise Support Manager
- Team alias for the account team: aws-moniepoint-team@amazon.com
