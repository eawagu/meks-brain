## Overview

Monify is a Moniepoint switch platform. Significant compliance uplift work was executed during the "David phase":

- Transaction monitoring (limit monitoring initiated by Damilari; specific monitoring by Awal)
- Centralized UBO register deployed
- Customer screening and risk rating completed, reflected on dedicated dashboard (with Loom)
- Cybersecurity compliance audit completed; first CBS report submitted
- Switch failure test successfully performed
- Fraud policy developed and implemented

## Ongoing Challenges

- KYC due diligence for legacy customers remains problematic
  - Active legacy customers still transacting
  - Inactive legacy customers unresponsive to documentation requests
  - Deactivation does not resolve past-activity inquiries (e.g., 2024 transaction issues for a 2025-deactivated account)
- Address verification: third-party vendor struggled with supplied addresses; BRN-led verification added as complement
- Loom adaptation: current Loom implementation is bank-focused, doesn't fit switch operations — new PM re-evaluating from scratch

## 2026 Direction

- Integrate with Moneypoint onboarding system to leverage their due-diligence processes
- Expand fraud tool coverage via Loom
- Anticipatory engagement with CBN
