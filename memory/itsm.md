IT Service Management (ITSM) is an industry-standard framework for designing, delivering, managing, and improving the way IT services are used within an organization. Common frameworks include ITIL, COBIT, and ISO/IEC 20000. Core processes include incident management, problem management, change management, configuration management, and service request fulfillment.
