## Summary

Skill for writing quarterly performance appraisals in [[TeamApt]]'s MOOS platform. Step-by-step workflow: gather inputs (OKRs, actuals, self-review, manager context), calculate attainment %, apply calibration rules (no OKRs=2.0 default, <90%=2.0, fact-check external claims with CSAT), write SBI-structured comments, identify growth areas, set SMART expectations.

## Key Points

- MOOS HR platform for performance management (also references Lattice)
- Rating scale 1.0–6.0 for "Dream Makers" (employees)
- Calibration rules: no OKRs defaults to 2.0, <90% attainment = 2.0 cap
- External factor claims must be fact-checked against CSAT data
- SBI framework (Situation-Behavior-Impact) for structured feedback
- SMART expectations for growth areas
- Certification timelines noted (12–18+ months for card schemes)
