## Overview

Hardware Security Module (HSM) is a dedicated cryptographic processor that secures card keys.

## In the MFB Card Stack

Deployed alongside [[Postillion]] in the [[Moniepoint MFB]] card production and authorisation flow.

## In the Phoenix Card Management System

The new platform-built [[Card Management System]] under [[Project Phoenix]] uses **Thales HSM** for cryptographic key management (KVP, CVK, IK, CAK, ECK, EMK, KWP).
