## Overview

Sister App Sensing is an onboarding optimization that detects whether the user installing a Moniepoint app already has a "sister" Moniepoint app installed (e.g., Moniepoint POS, Business, Personal) on the same device. When detected, the onboarding flow uses trust signals and shared credentials to streamline sign-up.

## Impact

Contributes to the 88.2% sign-up conversion rate by reducing friction for existing Moniepoint users expanding into additional Moniepoint products.
