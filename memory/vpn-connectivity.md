## Definition

**VPN Connectivity** is the network-layer prerequisite for [[TeamApt]]'s bank-deployed components in the [[Direct to Bank program]] pipeline — each bank partner requires a dedicated VPN tunnel for the bank-side AptPay deployment to communicate with TeamApt Cloud and back to the bank core systems. VPN setup is typically negotiated separately from API/SLA workstreams and frequently emerges as a long-tail blocker.
