## Overview

EMV Data Prep is the workflow of preparing EMV personalization data for issued cards. As of 2026-04-13, [[Taiwo Ogundipe]] reported SecureID returned the shared file because required fields (including the issue public key) need to be populated. The team is collaborating with the [[Visa]] scheme to generate the key for resubmission. Source: [[ATPP Daily Standup Meeting - 2026_04_13 15_54 WAT]].
