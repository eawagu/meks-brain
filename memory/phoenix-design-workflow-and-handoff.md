## Summary

Defines the AI-assisted design workflow under Project Phoenix, integrating Phoenix Design System with Claude Code + Figma MCP. Designer remains accountable for everything that ships; AI generates screens, designer refines and validates across seven dimensions before engineering handoff.

## Key Points

- Workflow: Change spec → AI generates screens → Designer reviews/refines → 7-dimension validation → Ready for dev → Engineering builds → Named version capture
- Seven handoff dimensions: Structural, Taxonomy, Documentation, Problem-design fit, Content design, Visual content, Completeness (100% AC path coverage)
- Figma page structure: In prod. | Ready for dev. | Work in progress | Playground | Cover
- Phoenix DS: 4-tier token system (Tier 1 base values → Tier 4 components); 9 documentation areas
- Design principles: Effortless clarity, Trust always, Human at core, For everyone, World-class, Built to last
