## Summary
Markdown export of the same Apr 1, 2026 meeting as [[Alignment Aptpay Suite Cowrys MFB 20260401 Gemini]]. Covers card network compliance, CMS deployment, two-track API integration (acquirer + issuer), and new direct debit scheme. The client confirmed use of existing prepaid cards via AfroGo CMS, enabling immediate progress on both integration tracks.

## Key Points
- Cowry's MFB closed-loop card requires card network membership (Visa, Mastercard, or AfriGo) for wide terminal acceptance
- TeamApt to share two document sets by Friday: switching API requirements and issuer processing infrastructure setup
- Daniel Ojinaka to initiate direct debit API conversation with Olamide immediately; 1–2 week implementation expected
- Tokenization feature available once TeamApt infrastructure deployed in client environment
- Cowry previously used Paystack for direct debit but did not push it due to security exposure concerns
- New TeamApt direct debit scheme supports Access Bank and GT Bank (~40% market share)
