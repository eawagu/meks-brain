## Summary

Captured on 2026-04-14. Documents the operating model [[Emeka Awagu]] communicated to [[Tracy Ojaigho]] for the CI&P takeover of the existing cards team inside [[Moniepoint MFB]]. Mirrors the pattern [[Frank Atashili]] and [[Alex Adeyemo]] are running elsewhere: existing engineers continue maintaining the current system while a strike team builds the new platform. Includes an explicit audit commitment to incorporate existing team output into the CI&P roadmap.

## Key Points

- Emeka told [[Tracy Ojaigho]]: CI&P is taking over the existing cards team in [[Moniepoint MFB]].
- Operating model chosen: **keep existing people maintaining the current system, while a strike team builds out the new [[Card Issuance & Processing Platform]].** Explicitly following the model [[Frank Atashili]] and [[Alex Adeyemo]] are using on their platforms.
- Commitment to audit everything the existing cards team has already built and incorporate it into the [[Card Issuance & Processing Platform]] roadmap — existing work is not discarded.
- Two-track transition structure: continuity track (legacy maintenance) + build track (new platform) running in parallel.
