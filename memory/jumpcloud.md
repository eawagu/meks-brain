## Overview
[[Jumpcloud]] is the access management solution deployed at [[TeamApt]].

## Role in IMS Controls
Per the [[TeamApt Management Review 19_04_2026]], Jumpcloud is listed as a control in the ISMS risk register for Application Server and Database assets (risks: unauthorized access, insider threats, code injection, lack of patching — rated Major). Alongside cloud-provider patching and secure-code training for developers.
