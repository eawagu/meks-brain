## Summary

Regulatory framework for card-not-present (CNP) web payments. Internet transfers are the largest channel by volume: 11.6B transactions H1 2024 (51.91% of e-payments).

## Key Points

- Mandatory 2nd-factor authentication (OTP, biometric, or approved method) for all CNP transactions
- 3D Secure via Access Control Server (ACS) at issuer is primary protocol
- Card scheme neutrality enforced; no card-present vs. card-not-present discrimination
- PA-DSS certification required for payment apps; PCI DSS for all card data handling
- Domestic processing mandate: Nigerian-issued cards must be processed/settled domestically
- Failed transactions reversed per 2020 reversal timelines
