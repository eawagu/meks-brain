## Summary
A meeting between [[TeamApt]] (AptPay Suite team) and [[Cowry's MFB]] (Touch and Pay) on Apr 1, 2026 covered card network compliance requirements, CMS deployment options, switching/acquiring API integration, and a new direct debit scheme offering instant activation. The client decided to use traditional prepaid cards and pursue tighter integration for reliability, while also expressing interest in the new direct debit APIs.

## Key Points
- Cowry's closed-loop card must be placed on a card network (Visa, Mastercard, or AfriGo) to work on external terminals
- TeamApt can deploy its CMS internally and connect its switch to Cowry's environment as issuer processor
- Two integration tracks proposed: (1) switch API for Cowry as acquirer; (2) TeamApt's transaction processing deployed in Cowry's environment for issuer processing
- Cowry already has a card management system via AfroGo — unlocks immediate progress on both tracks
- TeamApt switch is certified against 4 networks including AfriGo, Visa, Mastercard; has NIBSS alternative routes
- New direct debit scheme: instant activation, no 15-transfer breakage; supports Access Bank and GT Bank
- Card tokenization confirmed as a deliverable feature once TeamApt infrastructure is deployed
- Cowry interested in QR code-based payment via mobile; Daniel Ojinaka to assign someone to the integration
