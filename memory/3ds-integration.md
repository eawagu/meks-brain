## Overview

3DS (3-D Secure) integration is part of TeamApt's [[TPP|Third-Party Processing]] / card payments program. Used for cardholder authentication on card-not-present transactions.

## Status (per [[TeamApt MANCo Meeting - 31 March 2026]])

- **3DS integration with [[Mastercard]]: build complete**, deployment to production ongoing.
- **[[Visa Issuer Processing Certification]]:** opened, system configuration ongoing.
- **Mastercard + Visa dispute management certification:** commencing.
