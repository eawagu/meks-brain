## Summary

Detailed framework for writing quarterly performance appraisals using SBI framework, OKR calibration, and growth expectations.

## Key Points

- SBI framework: Situation-Behavior-Impact for structured feedback
- 90% OKR attainment = Achieved; 50-90% = Partially; <50% = Not Achieved
- Default 2.0 for no OKRs; CSAT cross-checks required
- 4.0+ requires above-and-beyond evidence
- SMART expectations for growth areas
