## Summary

MasterCard programs and services reference: Mastercard (credit/debit), Debit Mastercard, Maestro (international), Cirrus (ATM). Covers transaction types, merchant classification (MCC for interchange rates), program-specific Stand-In parameters, and services (contactless, ecommerce, cross-border, installment, MoneySend).

## Key Points

- Programs: Mastercard (credit/debit), Debit MC, Maestro, Cirrus
- MCC determines transaction rules and interchange rates
- Stand-In parameters vary by program
- Services: contactless, ecommerce, cross-border, installment, MoneySend
