## Summary

Merchant data quality standards: MID, DBA name, legal name, terminal ID, postal code, phone, email. MCC misclassification results in chargebacks, fines, or processing restrictions. Validation: postal code consistency, phone number validation, terminal inventory reconciliation. Fines $250-5000 per violation; acquiring suspension if not remediated.

## Key Points

- MCC accuracy critical: misclassification → chargebacks, fines, restrictions
- Validation: postal code consistency, phone, terminal reconciliation
- Remediation: corrective action plans, $250-5000 fines, suspension
- Required data: MID, DBA, legal name, terminal ID, postal, phone, email
