## Overview

CRM-based rewards program with A/B test groups (rewards+comms, rewards only, control).

## Findings (2026 UK Retreat Day 2)

- **Existing customers**: A/B tests show negligible retention/engagement lift.
- **New users (honeymoon rewards)**: clear incremental impact.
- Data credibility in question (CRM tool + data pipeline).
- Cashback liability ~₦1.9B total (personal + business) — low.
- 2.7M accounts cashed out rewards in last 7 days.
- Intent: engagement/stickiness, NOT treasury revenue from unclaimed float. [[Tosin Eniolorunda]] confirmed write-back of unclaimed liability.

## Gap Hypotheses

- Communication insufficient.
- Feature discoverability within app.
- Partial cash-out not supported (user-requested, backlog).
- Checkout integration proposed.

## Decisions

- Clarify or kill. Amir inherits ownership.
- Despite data ambiguity, cannot eliminate due to sales/field/morale pressure.
