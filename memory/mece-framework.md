## Overview

MECE = Mutually Exclusive, Comprehensively Exhaustive. McKinsey-originated framework for structuring information hierarchies.

## Principle

At any level of a presentation, items must:
- **Mutually exclusive** — no overlap or repetition between items.
- **Comprehensively exhaustive** — cover all relevant points at that level of abstraction.

Each item can have child items forming a pyramid structure. Style choice: open with the conclusion or build up to it — audience-dependent.
