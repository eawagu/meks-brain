## Overview
[[Wiz]] is a cloud security posture management tool used by [[TeamApt]] to inventory cloud assets on AWS and GCP.

## Role in IMS Controls
Per the [[TeamApt Management Review 19_04_2026]]: during the internal audit (ISMS A5.9), the team demonstrated Wiz for cloud assets (AWS/GCP) and the [[Nexus portal]] for computing assets. A clear inventory of computing and cloud assets is maintained with serial numbers and assigned users. Gap flagged: no inventory sighted for Supporting Utilities (CCTV, biometric scanners, software) — OFI to extend inventory coverage.
