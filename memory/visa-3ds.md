## Definition

**Visa 3DS** (Visa 3-D Secure) is the cardholder authentication protocol Visa uses for e-commerce transactions. Within [[TeamApt]]'s acquiring products track, Visa 3DS is being piloted as part of the broader [[Visa]] acquirer programme. See [[3DS Integration]] for the cross-scheme integration concept.
