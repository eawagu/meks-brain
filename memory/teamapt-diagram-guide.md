## Summary

Comprehensive reference for [[TeamApt]]'s C4 architecture diagram conventions in draw.io. Covers diagram levels (C1 System Context through C4 Code/Sequence), color coding for systems/people/containers, XML structure for elements, and cross-system patterns (three-tier deployment: TeamApt Cloud → Bank-Deployed → Bank Systems).

## Key Points

- C4 modeling standard with four levels: System Context, Container, Component, Code/Sequence
- Color-coded element types for consistency across diagrams
- XML structure conventions for draw.io elements
- Three-tier deployment pattern: TeamApt Cloud → Bank-Deployed → Bank Systems
- Multiple context diagrams per system (consolidated + partner/product variants)
- Cross-tab navigation conventions
