## Summary

Comprehensive specification for MasterCard reversals and chargebacks covering the full dispute lifecycle: authorization reversals (0400/0410 messages), chargeback initiation and reason codes, representment (issuer re-presents to acquirer), arbitration, and time limits. Large reference document (83KB).

## Key Points

- Authorization reversals via 0400/0410 message pairs
- Chargeback lifecycle: initiation → representment → arbitration
- Reason codes categorized by fraud, authorization, processing errors, cardholder disputes
- Time limits and windows for each stage of the dispute process
- Financial impact and liability shift rules
