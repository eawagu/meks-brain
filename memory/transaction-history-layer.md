## Overview

The Transaction History Layer is an internal Moniepoint service responsible for surfacing user transaction history in-app, and providing the substrate for dispute workflows, erroneous-transfer self-service, and analytics.

## Significance

Discussed in Day 2 Platform Performance session as a component supporting account payments reliability and self-service features (e.g., erroneous transfer self-service).
