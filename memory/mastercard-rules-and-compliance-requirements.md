## Summary

Comprehensive scheme rules: chargeback reason codes, authorization matching, amount verification, MCC appropriateness. Maximum chargeback rates 0.5-1.5% by program. Regulatory: PCI-DSS data security, GDPR cardholder privacy, AML/CTF monitoring.

## Key Points

- Chargeback reason codes: 4855, 4863, 4834, 4841, 4879, 4999
- Compliance: max chargeback rates, retrieval response timeframes, evidence retention
- Transaction rules enforce auth matching and merchant category appropriateness
