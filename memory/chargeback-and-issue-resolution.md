## Overview

**Chargeback and Issue Resolution** is an operations sub-team embedded within the [[MFB Cards Team]] at [[Moniepoint MFB]].

## Scope

- Chargeback processing

**Highest-priority AI automation target** for the CI&P strike team. Targets: automating dispute intake, routing, status communication.
