## Summary

Legal agreement where TeamApt signs up to Access Bank's Fund Transfer API for processing payouts for its customers. Access Bank deploys API service to facilitate TeamApt's payment offerings. Agreement covers roles and responsibilities, authorized accounts (debit restricted to prior written consent), indemnity and limitation of liability (mutual indemnification), data retention and audit rights, security breach management, termination conditions (breach remedy window), sub-processing, international transfers, data subject rights, and notices.

## Key Points

- Parties: Access Bank PLC (RC125384, Plot 14/15 Prince Alaba Oniru Street, VI, Lagos) and TeamApt Limited (RC1253771, 12 Wole Ariyo Street, Lekki Phase 1, Lagos)
- Purpose: TeamApt accessing Access Bank Fund Transfer API for payout processing
- Bank charges, set-offs, or administrative fees beyond payout execution require TeamApt's prior written consent
- Mutual indemnification clause (Section 13)
- Termination: Either party may terminate for unremedied breach
- Includes Data Protection addendum: data retention, audit rights, security incident management, international transfers, data subject rights
- Anti-corruption compliance: Corrupt Practices Act 2000, Money Laundering Act 2022, EFCC Act 2004, Advanced Fee Fraud Act 1995
