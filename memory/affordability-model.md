## Overview

The **Affordability Model** is a new credit-sizing model developed by [[Yuki]] for [[Moniepoint]], replacing the prior simplistic model based on turnover and business time. Uses more variables to produce better-calibrated loan sizes.

Alongside the model change, the lending organization introduced dedicated underwriters and a new approval flow for large loans; loans large enough to require CRO oversight get reviewed by a named employee before approval.
