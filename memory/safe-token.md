## Overview

Safe Token is the current OTP-based protocol used by [[Moniepoint MFB]] for secure card operations, handled by the [[Card OTP Service]]. OTPs are dispatched via MoniePoint's pre-whitelisted gateway.

## Extensibility

The Card OTP Service is designed to be extensible to additional protocols: [[Visa]] Token and 3D Secure / [[ACS]].
