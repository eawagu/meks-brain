## Overview

**CMS Manager** is an alias for [[Moniepoint MFB]]'s legacy internal Card Management System — the same system documented as [[Card Manager Service]]. The "CMS Manager" label was used in Project Phoenix MFB-Systems-Blindspot analysis to denote the MFB-built bridge (~12-person Cards Infrastructure team).

## Identity Resolution

CMS Manager = [[Card Manager Service]] (canonical title). See Card Manager Service for full scope, architecture, limitations, and Phoenix replacement plan.
