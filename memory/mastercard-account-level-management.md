## Summary

AMS file types: MCC102 (stand-in accounts), MCC103 (warning bulletins), MCC104 (local stoplists), MCC105 (payment cancellations), MCC106-111 (PAN mappings, enhanced value, product graduation, ATCs, PAN-PAR mappings). Stand-in processing allows transactions when issuer unavailable (acquirer assumes risk).

## Key Points

- 11 MCC file types for different account management operations
- Stand-in accounts: transactions proceed when issuer unavailable
- ALM Center Services: loyalty, prepaid, digital wallet integration
- File updates via IPM message format
