## Summary
Mar 30, 2026 all-hands for the Cards and Account team at [[Moniepoint]]/[[TeamApt]]. Key agenda: Smart Cache integration with invalid virtual account issue, 401 unauthorized error on Money Point NIBSS mandate authorization, and decision to deploy international payments (Mastercard/USD) to production behind a feature flag for internal testing.

## Key Points
- Smart Cache deployment complete but virtual accounts marked "invalid" during transfers; awaiting resolution from Smart Cache team
- Money Point virtual account: 401 unauthorized error on NIBSS mandate authorization; awaiting response from NIBS team after removing bearer token
- International payment (Mastercard + USD) to move to production behind feature flag for internal pilot testing
- Visa international payments blocked by configuration key issues; Mastercard + USD first
- Compliance feedback from Eddie Fami (FDR, merchant transaction limits) not a blocker for testing
- Pre-deployment actions required: FAQ updates, CS walkthrough, newsletters to merchants, new report format rollout
- Ankit Kushwaha to schedule stakeholder calls for all 3 projects: Smart Cache, Money Point VA, USD transactions
- New transaction report format will show currency identification
