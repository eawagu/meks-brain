## Overview

User Acceptance Testing (UAT) is the testing phase in which a bank or partner counterparty exercises an integration end-to-end against an agreed-upon scope before production cutover. Within [[TeamApt Bank Integration]], UAT typically follows VPN provisioning, SLA signing, API configuration, and IP whitelisting, and is the immediate gate to go-live.
